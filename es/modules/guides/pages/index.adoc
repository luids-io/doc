| Guía | Descripción

| xref:guide-full-dns-sinkhole:index.adoc[Sumidero DNS paso a paso]
| Guía completa de configuración de un sumidero DNS paso a paso.

| xref:guide-basic-dns-sinkhole:index.adoc[Sumidero DNS básico]
| Guía rápida de configuración de un sencillo sumidero con el que bloquear publicidad, phising y malware.

| xref:guide-advanced-dns-sinkhole:index.adoc[Sumidero DNS avanzado]
| Guía rápida de configuración de un sumidero con el que bloquear publicidad, phising y malware. Este sumidero además utiliza la API Google Safebrowsing, archiva las peticiones DNS realizadas, genera eventos y dispone de una interfaz web de visualización del archivo.

| xref:guide-basic-dns-rblzone:index.adoc[Zona DNS RBL compuesta]
| Guía rápida de configuración de una sencilla zona DNS RBL compuesta para su uso como sistema antispam.

| Configuración de un firewall iptables para bloquear Botnets.

| xref:guide-basic-nfqueue-resolv:index.adoc[Cortafuegos integrado con DNS]
| Configuración de un firewall iptables integrado con una caché de resoluciones DNS para bloquear aquellas conexiones que no hayan pasado por el proceso de resolución de nombres.

| xref:guide-basic-nfqueue-whitehole:index.adoc[Cortafuegos integrado con sumidero DNS]
| Configuración de un firewall iptables junto a un sumidero DNS de lista blanca para que bloquee todas aquellas conexiones provenientes de equipos de nuestra red dirigidas a dispositivos que no hayan sido resueltos previamente por nuestro servidor DNS.
