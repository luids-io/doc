TIP: Si está instalando en una Raspberry debe definir la arquitectura al lanzar el instalador con `sudo ARCH=arm64 ./installer.sh`. Si desea instalar el software en alguna otra arquitectura, las arquitecturas disponibles son: `amd64`, `arm`, `arm64`, `mips`, `mips64`, `mips64le`, `ppc64` y `s390x`.

Además en esta guía utilizaremos `iptables-persistent` para establecer las reglas del cortafuegos. Lo instalaremos vía `apt`.

[source,bash]
----
sudo apt install iptables-persistent
----
